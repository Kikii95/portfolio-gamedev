### Zero-overhead Design

The monitor is designed to have minimal impact on the system it's monitoring. Metrics are collected efficiently using system APIs, and the TUI renders only when data changes.

### GPU Integration

NVIDIA GPU monitoring is implemented via the NVML library, providing real-time access to GPU utilization, memory, temperature, and power consumption.
