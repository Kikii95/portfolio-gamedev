## ðŸš€ Key Features

### Terminal Core
- **Multi-tab Support**: Unlimited sessions with drag-and-drop reordering
- **Split Panes**: Horizontal and vertical splits with resizable panels
- **Multi-window**: Detach tabs into separate windows
- **Session Restore**: Automatic restoration on restart
- **Search**: Full text search with Ctrl+F

### Developer Tools
- **Git Panel**: Branch management, staging, commits, push/pull
- **Docker Panel**: Container and image management
- **SSH Manager**: Save and quick-connect to SSH hosts
- **Project Switcher**: Navigate projects by category
- **Quick Commands**: Save and execute frequent commands

### UI/UX
- **6 Themes**: Catppuccin, Dracula, Nord, One Dark, Gruvbox, Tokyo Night
- **Command Palette**: Quick access with Ctrl+Shift+P
- **Quake Mode**: Dropdown terminal with customizable hotkey
- **Notifications**: Alert when long commands complete

## ðŸ”§ Technical Highlights

### Performance

Lightweight compared to Electron alternatives: ~15MB installer, ~80-120MB RAM. The Tauri backend handles shell spawning and IPC, while React handles the UI efficiently.

### Shell Integration

Supports all installed WSL distributions (auto-detected), PowerShell, and CMD. The project switcher integrates with my `~/projects/` directory structure.
