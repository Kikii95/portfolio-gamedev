# Physics Engine

Moteur physique temps réel implémentant dynamique rigid body, détection de collisions et résolution par contraintes.

## Contexte

Projet école GTech3, en cours. Équipe de 3, rôle : développeur systèmes physiques.

La simulation physique est fondamentale dans les jeux modernes. Ce projet implémente les concepts core from scratch.

## Stack

- C++
- Raylib (visualisation)
- Algèbre linéaire custom
- CMake

## Fonctionnalités

- Dynamique rigid body : masse, vélocité, forces, torque
- Détection collisions : broad phase (spatial hashing) + narrow phase (GJK/EPA)
- Réponse collisions : résolution par impulsion
- Contraintes : joints distance, charnière, slider
- Visualisation debug : affichage état physique temps réel

## Points Techniques

Intégration physique :
- Euler semi-implicite pour la stabilité
- Velocity Verlet pour la précision
- Timestep fixe avec interpolation

Pipeline de collisions multi-phase :
- Partitionnement spatial pour broad phase
- Separating Axis Theorem pour formes convexes
- Génération et cache de points de contact
