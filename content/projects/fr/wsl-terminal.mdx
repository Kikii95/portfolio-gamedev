# WSL Terminal

Émulateur de terminal moderne pour WSL2, optimisé pour le développement.

## Contexte

Projet perso démarré en décembre 2025. Développé en solo sur 2 semaines.

Windows Terminal est correct, mais je voulais quelque chose avec une meilleure intégration développeur : panneaux Git, gestion Docker, navigation projet, UI personnalisable — sans la lourdeur d'Electron.

## Ce que ça fait

**Terminal** :
- Multi-onglets avec drag-and-drop
- Split panes horizontal/vertical redimensionnables
- Multi-fenêtres (détacher des onglets)
- Restauration de session automatique

**Outils dev** :
- Panneau Git : branches, staging, commits, push/pull
- Panneau Docker : conteneurs et images
- Manager SSH : sauvegarde et connexion rapide
- Switcher de projets par catégorie

**Interface** :
- 6 thèmes (Catppuccin, Dracula, Nord, One Dark, Gruvbox, Tokyo Night)
- Font ligatures (Fira Code, JetBrains Mono)
- Palette de commandes Ctrl+Shift+P
- Mode Quake (dropdown)
