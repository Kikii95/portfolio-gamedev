# Audio Editor

Éditeur audio desktop pour fichiers WAV avec effets DSP, visualisation waveform et traitement temps réel.

## Contexte

Projet école GTech3, 6 semaines. Équipe de 3, rôle : développeur DSP et moteur audio.

Exploration des fondamentaux du traitement de signal numérique via une application d'édition audio pratique.

## Cahier des Charges

Objectifs du projet :
- Parser et lire des fichiers WAV (PCM)
- Affichage waveform en temps réel
- Implémenter des effets DSP (EQ, compression, reverb, delay)
- Édition non-destructive avec preview

Contraintes :
- API audio Windows (XAudio2 ou WASAPI)
- Formats WAV 8/16/24-bit, mono/stereo
- Latence < 50ms pour la lecture
- Interface utilisateur fonctionnelle

## Stack

- C++
- XAudio2 (API audio)
- Parser WAV custom
- Win32 / ImGui (UI)

## Fonctionnalités

- Import/export WAV (PCM complet)
- Affichage waveform temps réel
- Effets DSP : EQ, compression, reverb, delay
- Édition non-destructive avec preview de la chaîne d'effets
- Outils de sélection : cut, copy, paste de régions audio

## Points Techniques

Parser WAV custom gérant :
- Parsing header RIFF
- Formats multiples (8/16/24-bit, mono/stereo)
- Navigation chunks et métadonnées

Effets audio implémentés from scratch :
- Biquad filters pour bandes EQ
- Compresseur RMS
- Delay line avec feedback
- Bases de reverb à convolution
