# Audio Editor

Éditeur audio desktop pour fichiers WAV avec effets DSP, visualisation waveform et traitement temps réel.

## Contexte

Projet école GTech3, 6 semaines. Équipe de 3, rôle : développeur DSP et moteur audio.

Exploration des fondamentaux du traitement de signal numérique via une application d'édition audio pratique.

## Cahier des Charges

Consignes école :
- Parser et lire des fichiers WAV
- Affichage waveform
- Implémenter au moins 2 effets DSP
- API audio Windows, latence < 50ms

## Stack

- C++
- XAudio2 (API audio)
- Parser WAV custom
- Win32 / ImGui (UI)

## Notre Implémentation

Au-delà des consignes (2 effets demandés), nous avons poussé le projet :

- **4 effets DSP complets** : EQ paramétrique, compression RMS, reverb à convolution, delay avec feedback
- **Édition non-destructive** : preview de la chaîne d'effets avant application
- **Outils de sélection avancés** : cut, copy, paste de régions audio avec précision sample
- **Support formats étendus** : 8/16/24-bit, mono/stereo (au-delà du 16-bit demandé)

## Points Techniques

Parser WAV custom gérant :
- Parsing header RIFF
- Formats multiples (8/16/24-bit, mono/stereo)
- Navigation chunks et métadonnées

Effets audio implémentés from scratch :
- Biquad filters pour bandes EQ
- Compresseur RMS
- Delay line avec feedback
- Bases de reverb à convolution
