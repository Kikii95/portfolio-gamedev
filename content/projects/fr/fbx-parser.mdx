# FBX Parser

Parser custom pour le format binaire FBX, permettant l'import de modèles 3D, animations squelettiques et hiérarchies de scène.

## Contexte

Projet école GTech3, 5 semaines. Équipe de 3, rôle : implémentation du parser binaire.

Comprendre les formats de fichiers 3D est essentiel en dev jeux. Ce projet plonge dans la spécification binaire FBX.

## Cahier des Charges

Objectifs du projet :
- Parser le format binaire FBX 7.x
- Extraire mesh, matériaux et données d'animation
- Reconstruire la hiérarchie de scène
- Gérer la décompression zlib

Contraintes :
- Pas d'utilisation du SDK FBX officiel
- Support des fichiers compressés et non-compressés
- Export vers format intermédiaire simple
- Documentation du format parsé

## Stack

- C++
- CMake
- FBX Binary (7.x)
- zlib (compression FBX)

## Fonctionnalités

- Parsing format binaire : traversée des nœuds FBX bas niveau
- Extraction mesh : vertices, normales, UVs, indices
- Support matériaux : propriétés basiques
- Hiérarchie de scène : relations parent-enfant des transforms
- Données animation : extraction keyframes

## Points Techniques

Structure binaire FBX :
- Arbre de nœuds
- Tableaux de propriétés avec types variés
- Compression zlib optionnelle
- Encodage custom pour strings et arrays

Extraction de données :
- Géométrie (vertices, polygones, normales)
- Mappings UV (canaux multiples)
- Poids de bones pour le skinning
- Courbes d'animation
