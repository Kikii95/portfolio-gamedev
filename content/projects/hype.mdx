# Hype

Repo privé — Projet startup

Application mobile pour découvrir les événements et expériences autour de soi.

## Contexte

Projet startup démarré en juin 2025. Rôle : Full-stack Developer. Phase MVP.

Problème : trouver des trucs à faire ne devrait pas nécessiter de checker 10 apps différentes.

## Stack

- React Native, Expo (mobile)
- Next.js, TypeScript (web admin)
- REST API (backend)
- PostgreSQL
- Monorepo pnpm

## Fonctionnalités

- Découverte intelligente basée sur préférences et localisation
- Mises à jour temps réel des événements
- Intégration sociale : voir les intérêts des amis
- Cross-platform : iOS et Android natifs
- Dashboard web pour les organisateurs

## Points Techniques

Architecture monorepo avec pnpm, partage de code entre app mobile, dashboard web et backend. Cohérence garantie, duplication minimale.

Services de géolocalisation pour le moteur de découverte, événements pertinents selon proximité et préférences de déplacement.
