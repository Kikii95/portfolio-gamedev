# LLM Automation Platform

Repo privé — Projet professionnel

Plateforme SaaS permettant de créer des workflows d'automatisation alimentés par des LLM, sans écrire de code.

## Contexte

Projet professionnel démarré en septembre 2025. Rôle : Lead Developer. Équipe restreinte.

La plateforme permet à des utilisateurs non-techniques de construire des workflows d'automatisation complexes avec des LLM, avec fiabilité et sécurité enterprise.

## Stack

- Python 3.11+, FastAPI (backend)
- LangGraph, LangChain (orchestration IA)
- TypeScript, React (frontend)
- PostgreSQL
- Docker, Redis (infra)

## Fonctionnalités

- Support multi-providers : Claude, OpenAI, Ollama (local)
- Builder visuel de workflows drag-and-drop
- Intégration RAG pour bases de connaissances custom
- Système de plugins extensible
- Sécurité enterprise : RBAC, audit logs

## Points Techniques

Architecture agents avec LangGraph pour des workflows stateful multi-étapes. Chaque agent peut utiliser des outils, maintenir un contexte de conversation et gérer des arbres de décision complexes.

Abstraction multi-providers : interface unifiée permettant de switcher entre Claude, GPT-4 et modèles Ollama locaux selon les besoins.

Métriques : 77% de couverture de tests, 1296 tests.
