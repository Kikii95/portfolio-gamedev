### Serverless Architecture

The entire app runs client-side with no backend server. Authentication uses GitHub OAuth, and all data operations go directly through the GitHub API. Your token is stored in an encrypted JWT cookie.

### Security Model

| Aspect | Implementation |
|--------|----------------|
| Auth | GitHub OAuth |
| Storage | JWT cookie (no server database) |
| Access | Only configured repos |
| Data | Never stored on our servers |
