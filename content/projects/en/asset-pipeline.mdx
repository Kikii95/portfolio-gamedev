# Asset Pipeline Engine

Robust asset management system for game engines: loading, caching, hot-reload and dependency resolution.

## Context

School project GTech3, 5 weeks. Team of 3, role: core pipeline architect.

Modern games require sophisticated asset management. This project implements a production-ready pipeline.

## School Requirements

- Implement a resource loading system
- Automatic memory management
- Hot-reload support
- Modern C++, documentation, unit tests

## Our Implementation

Beyond requirements:

- **Async loading with callbacks**: visible asset prioritization
- **Reference counting**: prevents memory leaks
- **Smart hot reload**: file watcher + dependency graph
- **LZ4 compression**: 60% bundle size reduction
