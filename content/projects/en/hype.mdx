# Hype

Private repo — Startup project

Mobile app to discover events and experiences around you.

## Context

Startup project started in June 2025. Role: Full-stack Developer. MVP phase.

Problem: finding things to do shouldn't require checking 10 different apps.

## What it does

**Discovery**:
- Personalized feed based on preferences and location
- Filters: category, distance, date, price
- Smart recommendations

**Social**:
- See friends' interests
- Share and save events
- Real-time notifications

**Organizers**:
- Web dashboard to create/manage events
- Attendance analytics
- Targeted promotion
